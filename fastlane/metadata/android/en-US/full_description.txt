<p><b>⚠️ Alpha Stage</b></p>
<p>Expect bugs, instability, and incomplete features.</p>

<p>Closed-source launchers are locking features behind paywalls and adding trackers. My goal is to offer powerful features with no compromises on privacy.</p>

<p><b>YAGNI ("You Aren't Gonna Need It")</b> is a principle which arose from extreme programming (XP) that states a programmer should not add functionality until deemed necessary.</p>

<p><b>Features</b></p>
<ul>
<li>No fixed grid size. You can set the grid width and height to any value you want.</li>
<li>Each grid item is fully customizable:
  <ul>
    <li>Size and rotation</li>
    <li>Background color</li>
    <li>Icon size</li>
    <li>Text and label size</li>
    <li>Label color</li>
    <li>Show or hide icons and labels</li>
  </ul>
</li>
<li>Infinite scrolling</li>
<li>Multiple dock pages</li>
<li>Adjustable dock height</li>
<li>Custom dock grid size</li>
</ul>

<p>Explore and discover more features, only ~2MB</p>
